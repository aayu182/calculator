# Calculator CLI App
A simple command - line calculator built using Python.
## Features:
- Addition , Subtraction , Multiplication , Division
- Input Validation
- Loop until user exits
How to Run
1 . Clone the repository:
  Bash
  git clone
  https://github.com/aayu182/calculator/edit/main/README.md
2 . Navigate to the folder:
  Bash
  cd Calculator-CLI
3 . Run the script:
  Bash
  python calculator.py
Learning Outcomes
 Understanding of Python functions
 Handling user input using input()
 Implementing loops and conditionals
 Building a basic CLI program
